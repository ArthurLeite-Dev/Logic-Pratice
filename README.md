# C# Logic Programming - Practical Exercises

This repository contains a collection of practical logic exercises developed using **C#** and **Windows Forms** (.NET Framework). 

> [!IMPORTANT]
> **Project Recovery Note:** This source code was successfully recovered and refactored from a compiled assembly using **ILSpy** after the original source files were lost.

## Included Exercises:

The application features a main dashboard that provides access to four distinct logic challenges:

1. **Number Range Generator:** Lists all integers between two user-selected values (supporting both ascending and descending order).
2. **Real-time Text Formatter:** A tool that converts input text to uppercase instantly as the user types.
3. **Dynamic UI Styler:** Allows the user to interactively change the application's background color using radio buttons.
4. **Snack Shop Calculator:** A simple Point of Sale (POS) simulator that calculates the total price of a selected order, featuring a variety of items.
